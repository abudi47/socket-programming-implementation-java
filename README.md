A simple client-server communication example using Java Sockets.

Files
    client.java: Sends a message to the server.
    Server.java: Listens for a client connection and responds.
Requirements
    JDK 8 or higher.
How to Run
1) Compile the Code:
    
    javac Server.java
    javac client.java
2) Run the Server:
    java Server
3) Run the Client:
    java client
